# RL Reading Group

Welcome to the RL Reading Group.

This repository is used to record the papers selected for our weekly reading group.  
Please add the paper you choose each week before the meeting.

## How to Add Your Paper

Please add your selected paper to the table above using the following format:

```markdown

| YYYY-MM-DD | Your Name | Paper Title | Venue / Year | [Paper](URL) | Brief notes |

```
